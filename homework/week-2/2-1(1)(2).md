# 2-1(2)
    由算数平均式得
        (n1 + n2 + …… +na) / a  >=  (n1 * n2 * …… * na) ^ (1 / a)
    将n按长度x分为a段
    x^a = (x^(1/x))^n  (n固定,x未知)
    求(x^(1/x))最大值
        y = x ^ (1/x)
        lny = (lnx)/x
        当x取e = 2.71……得最大值
        且x = 3 比 x = 2更大
    所以n = 3a + b
    当b = 0时，全取3，a = n/3
    当b = 1时，留一个3和1组合变成2*2，其余取3
    当b = 2时，2个2，其余取3
